What I can do？

我觉得设计师的价值在于不断创新，始终追求以更好的方式解决实际问题。不管是从事平面设计、UI设计还是体验设计，都需要不断的探索问题、解决问题，并在过程中提升自己。

01 / Mobile App

移动端 & 小程序

专注于 iOS 和 Android 的原生体验设计。通过深入的用户流程分析 (User Flow) 和高保真原型图，打造符合用户直觉的移动端产品。

Deliverables 交付成果
  • User Flow Chart - 用户体验路径图
  • Hi-fi Prototype - 高保真交互原型
  • Design System - 移动端设计规范
界面设计 原型设计 设计系统 微交互动效设计（AE）
02 / Website

网站 & 落地页设计

我创建视觉效果、以用户为中心的网站和落地页，简化交互，设计并开发响应式网站。注重微交互动效和 SEO 优化，确保网站既好看又好用。

Deliverables 交付成果
  • Responsive Layout - 响应式网页布局
  • Landing Page - 营销落地页设计
  • Web Component - 网页组件库
Framer / Webflow 设计系统/Flex布局 Cursor / VS Code 微交互动效设计(剪映)
03 / Brand

标志 & 品牌

一个标志不仅仅是装饰。它是你品牌的入口。我设计清晰、易记的标志，传达你的价值观并建立即时识别度。

Deliverables 交付成果
  • Primary Logo - 品牌主标志
  • Visual Identity - VI 视觉识别手册
  • Marketing Assets - 社交媒体延展物料
Logo 设计 Visual 定义 设计营销资产
04 / Icon

图标 & 插画

设计富有表现力的图标和插画，增添个性，引导用户，并通过视觉丰富品牌的故事讲述。

Deliverables 交付成果
  • Icon Set - 定制化图标库
  • Vector Illustration - 矢量商业插画
  • AIGC辅助设计 -图标、场景设计
Illustrator/Ps/Ai Procreate
